## 料金プラン

Factoryは、Standard Tokenを通じて使用量を測定しています。キャッシュされたトークンは、Standard Tokenの10分の1の料金で請求されます（キャッシュトークン10個 = Standard Token 1個）。2つのサブスクリプションプランを提供しています：

| プラン     | Standard Tokens / 月                          | 月額料金      |
| -------- | --------------------------------------------- | ------------- |
| **Free** | BYOK                                          | $0            | 
| **Pro**  | 2,000万                                       | $20           |
| **Max**  | 2億                                           | $200          | 

超過分は、Factory Standard Token 100万あたり$2.70で請求されます。

## トークンについて考える

参考として、GPT-5.3-Codexを0.7×の倍率で使用し、一般的なキャッシュ比率である4〜8倍を併用すると、実効的なStandard Token使用量は、オンデマンドでの直接呼び出しよりもはるかに効率的になります。非常に高価なモデルを頻繁に切り替えたり、キャッシュが無効になるほど頻繁にモデルをローテーションしたりすると、その効果は低下しますが、ほとんどのワークロードでは、個別のモデルプロバイダーから直接容量を購入する場合と比較して、大幅に高い使用上限を実現できます。私たちの目標は、トークンの計算を気にすることなくワークロードを実行していただくことです。プランは、一般的な使用パターンが類似の直接提供サービスを上回る性能を発揮するよう設計されています。
