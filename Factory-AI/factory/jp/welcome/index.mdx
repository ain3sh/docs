Factoryは、あなたが作業するあらゆる場所で使えるAIネイティブなソフトウェア開発プラットフォームです。このページで適切な出発点を選び、ワークフローに合った機能、設定、エンタープライズ向けドキュメントへ進んでください。

## 利用環境から始める

<CardGroup cols={2}>
  <Card title="Droid CLI" icon="square-terminal" href="/jp/cli/getting-started/quickstart">
    ターミナルでDroidを実行して、数分で作業を始められます。
  </Card>
  <Card title="Factory App" icon="browser" href="/jp/web/getting-started/quickstart">
    Factory Appをダウンロードして、Droidをローカルマシンに接続します。
  </Card>
  <Card title="Droid Exec" icon="terminal" href="/jp/cli/droid-exec/overview">
    CI/CDや自動化ワークフローでDroidを非対話的に実行します。
  </Card>
  <Card title="エンタープライズ" icon="building" href="/jp/enterprise">
    高セキュリティ環境でDroidを導入、管理、監視します。
  </Card>
</CardGroup>

## 主要機能を見る

<CardGroup cols={2}>
  <Card title="Factory Missions" icon="diagram-project" href="/jp/cli/features/missions">
    構造化された複数ステップのワークフローで、より大きなプロジェクトを計画して実行します。
  </Card>
  <Card title="ローカルコードレビュー" icon="code-branch" href="/jp/cli/features/code-review">
    `/review`ワークフローと再利用可能なガイダンスで、ローカルのコード変更をレビューします。
  </Card>
  <Card title="Droid Control" icon="toolbox" href="/jp/cli/features/droid-control">
    QAと検証のために、ターミナル、ブラウザ、デスクトップアプリを自動化します。
  </Card>
  <Card title="Agent Readiness" icon="chart-line" href="/jp/web/agent-readiness/overview">
    リポジトリの成熟度を測定し、自律性の進捗を継続的に追跡します。
  </Card>
  <Card title="Droid Computers" icon="computer" href="/jp/cli/features/droid-computers">
    リモートでのDroid作業向けに、永続的なローカルまたはクラウドマシンを接続します。
  </Card>
  <Card title="クラウドテンプレート" icon="cloud" href="/jp/web/machine-connection/cloud-templates/index">
    開発環境に合った事前構成済みのクラウド環境を起動します。
  </Card>
</CardGroup>

## Droidを設定して拡張する

<CardGroup cols={2}>
  <Card title="設定" icon="gear" href="/jp/cli/configuration/settings">

  </Card>
  <Card title="独自キーの使用（BYOK）" icon="key" href="/jp/cli/byok/overview">

  </Card>
  <Card title="フック" icon="wrench" href="/jp/cli/configuration/hooks-guide">
