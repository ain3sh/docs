<Update label="March 5" rss={{ title: "CLI Updates", description: "CJK internationalization, GPT-5.4 model updates, and --mission flag for droid exec" }}>

  * **Git 設定の更新** - 明示的に要求した場合に Git 設定を更新可能になりました
  * **BYOK Anthropic thinking** - カスタム Anthropic モデルの適応的思考サポートを改善
  * **スマートリトライ処理** - プロバイダー容量エラーのリトライ遅延を改善

  * **MCP ツールレンダリング** - Web アプリでの MCP ツール結果表示を改善（app）

  * **空のモデル応答** - 空のモデル応答をエラーを引き起こす代わりに適切に処理するように修正
  * **思考の回復** - 推論中の思考シグネチャ検証エラーからの自動回復
  * **スペックモードナビゲーション** - スペックモードでの自律メニューからの戻りナビゲーションを修正
  * **ミッションセッションクリーンアップ** - 提案されたミッションを終了する際に新しいセッションが正しく開始されるように修正
  * **Gemini エラーハンドリング** - Gemini モデルでの認証と支払いエラーのエラーハンドリングを改善
  * **プロファイル表示** - 名前フィールドが不完全なアカウントのユーザープロファイル解析を修正
