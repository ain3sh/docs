## Managing Mirrors

- Update `mirrors.json` (it follows [`mirrors.schema.json`](./.github/mirrors.schema.json)) directly in the GitHub web editor or via PR to add/remove mirrors.
- Each entry requires `owner`, `repo`, `branch`, and `docsPath`.
- The daily `Sync Mirrors` workflow clones every configured docs path, refreshes this repo, and rewrites this README.
- To sync immediately, run `python3 .github/scripts/sync_mirrors.py` locally (use `--only owner/repo` to target a single mirror).
